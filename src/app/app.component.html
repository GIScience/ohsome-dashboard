Technology

The “ohsome” OpenStreetMap History Analyzer Platform is based on technology developed by HeiGIT.

The source code for this dashboard is open source (GPL-3.0) and available on GitHub.

Data Analyses are powered by:

OSM History Stats: ohsome API (AGPL-3.0)
OSM Quality Analyses: ohsome quality API (AGPL-3.0)
